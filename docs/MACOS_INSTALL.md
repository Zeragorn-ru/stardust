# Установка StarDust на macOS

Гайд для игроков, которые скачали лаунчер с [GitHub Releases](https://github.com/Zeragorn-ru/stardust/releases).

## 1. Скачайте DMG

1. Откройте [релизы Stardust](https://github.com/Zeragorn-ru/stardust/releases).
2. Найдите последний тег `vX.Y.Z` (например `v0.7.24`).
3. Скачайте файл **`StarDust_X.Y.Z_universal.dmg`** — он подходит и для Intel, и для Apple Silicon.

## 2. Установите приложение

1. Откройте скачанный `.dmg` двойным кликом.
2. В окне установщика перетащите **StarDust.app** в папку **Applications** (как на фоне DMG).
3. Извлеките диск (ПКМ по томе → **Извлечь**) или перетащите в корзину.

## 3. Первый запуск (важно)

Лаунчер пока **не подписан** платным сертификатом Apple ($99/год). macOS может показать:

> **«StarDust» Not Opened** — Apple could not verify…

Это нормально для нашей сборки. Сделайте так **один раз**:

1. Откройте **Finder → Программы**.
2. Найдите **StarDust**.
3. **Правый клик** (или Control+клик) → **Open** / **Открыть**.
4. В диалоге снова нажмите **Open** / **Открыть**.

После этого macOS запомнит разрешение, и лаунчер будет открываться обычным двойным кликом.

Альтернатива: **Системные настройки → Конфиденциальность и безопасность → Open Anyway**.

## 4. Полный гайд в DMG

В окне DMG есть файл **«Установка»** — двойной клик откроет эту страницу в браузере.

## 5. Обновления

Установленный лаунчер сам проверяет [GitHub Releases](https://github.com/Zeragorn-ru/stardust/releases) и предлагает обновление в настройках (**Обновления**).

## 6. Сборка из исходников (NixOS / без DMG)

На NixOS AppImage не запускается — собирайте из исходников: см. [README](../README.md) и `flake.nix`.

## Проблемы?

- [Issues на GitHub](https://github.com/Zeragorn-ru/stardust/issues) — опишите версию macOS и скрин ошибки.
- Убедитесь, что скачали **universal** DMG, а не Windows `.exe` или Linux `.deb`.
